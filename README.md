【PaxPowerGloveAllHard】

■目的

 ファミコン拡張コントローラー（パワーグローブ）から各種ハードを操作する


■最終目標

　・FC → SFC（SGB2対応）

　・FC → GC（GBプレイヤー）

　・FC → 他ハード展開


■現状
　・通常SFC：動作OK（ゼルダの伝説神々のトライフォース、スーパーマリオワールド、星のカービィスーパーデラックスは動いた）

　・SGB2：無反応（精度不足）

　　※他には聖剣伝説２のロード以降、マリオコレクションの各ゲーム開始以降なをも動かない（メニューなどは動く）

　※詳細は docs/ を参照


■開発環境

　・Raspberry Pi Pico

　・Arduino IDE 1.8.19

　・KiCad


■重要ポイント

　・SGB2は非常に厳密なタイミング要求あり

　・ソフトのみでは限界 → ハード同期が必要
